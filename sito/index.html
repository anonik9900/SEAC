About SEAC

SEAC is a lightweigth tool devolved in Ruby, it allow you to edit, create and save some type of .ini file. The current version is in BETA [For Dev Only], this software use the inifile library to read and write all .ini files for Windows. SEAC use a config file called settings.ini (only for Windows PC), usefull to change the software language.

Compatibilty:

SEAC is designed and programmed for Windows PCs, but is executable on linux with the appropriate change (change the .ini settings system and compile to be runnable on Linux Darwin Kernel).
Windows (Windows 10,8.1,8,7 - 64bit), Linux (Ubuntu 16.x,Mint 10.x,Debian All).
